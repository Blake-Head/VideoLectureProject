# Physics 1 Lecture Videos

This folder is to contain animations for the University Physics 1 (PHYS 2425) and College Physics 1 (PHYS 1401) courses.  While several topics span the two, some notable differences. 

- 2425 has a calculus 1 *co-requisite*.  While the course (as of Summer 2026) contains very little calculus, it does appear a bit near the end. 1401 is algebra-based, and thus should have no mention of calculus in their lectures. 

- Students enrolle in 1401 are primarily health-science majors.  Examples should include more connections to pysiology.
